# Deploy vLLM-Ascend with MindIE-PyMotor

## Overview

[MindIE-PyMotor](https://gitcode.com/Ascend/MindIE-PyMotor) provides one-click deployment for **prefill–decode (PD) disaggregation** and **PD colocation** on Ascend NPUs. It combines high-performance scheduling and load balancing to build scalable, highly available inference services on top of **vLLM-Ascend**.

For PD disaggregation concepts, see [Using Volcano Kthena](./using_volcano_kthena.md) as an alternative orchestration option.

## Getting Started

For step-by-step deployment instructions, refer to the [MindIE-PyMotor User Guide](https://gitcode.com/Ascend/MindIE-PyMotor/blob/master/docs/zh/user_guide/README.md).
